## 📌 Problem Statement

Organizations maintain large volumes of employee data including department, experience, salary, and location. However, without proper analysis, this data cannot provide useful insights.

This project aims to analyze employee salary data to understand salary distribution across departments and evaluate how experience influences salary levels, helping management in better decision-making.

---

## 💡 Proposed Solution

This project follows a multi-tool approach for analysis and visualization:

* SQL is used for structured data querying
* Excel is used for summarization using Pivot Tables
* Python is used for data analysis and visualization

The system extracts key insights and presents them in a simple and clear format.

---

## 🚀 Features

* Department-wise salary analysis
* Experience vs salary relationship
* Highest-paid employee identification
* Average salary calculation
* SQL-based querying
* Excel Pivot Table summarization
* Data visualization (Histogram, Bar Chart, Scatter Plot)

## 📊 Key Insights

* Finance department has the highest average salary
* HR department has comparatively lower salaries
* Average salary is around **₹53,000 – ₹55,000**
* Employees with more experience tend to earn higher salaries
* Positive correlation exists between experience and salary
